# 🎭 FantasyToca

## 📖 Sobre o projeto

A **FantasyToca** é um marketplace de fantasias desenvolvido como projeto acadêmico do curso de **Sistemas de Informação na UNINOVE**.

O projeto tem como objetivo criar uma plataforma digital para facilitar a divulgação e navegação de fantasias, proporcionando uma experiência simples, moderna e intuitiva para os usuários.

## 🚀 Funcionalidades

* Catálogo de fantasias
* Interface responsiva
* Navegação intuitiva
* Organização de produtos
* Experiência focada no usuário

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
